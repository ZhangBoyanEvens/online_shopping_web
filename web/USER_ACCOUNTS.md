## Usage

1. Open `index.html` in your web browser
2. Click on the "Login" tab
3. Enter any of the email addresses and corresponding passwords from the table above
4. Click "Login" to access the shopping system

## Notes

- All passwords are stored in plain text in localStorage (for demo purposes only)
- In a production environment, passwords should be hashed
- You can also create new accounts using the "Sign Up" tab
- All user data is stored in browser localStorage
